## 課題91：Renderアカウント作成

Renderのアカウント登録を行い，ダッシュボード画面にログインできることを確認する

---

### 登録手順
#### ① Render公式サイトへアクセス

➡ https://render.com

- 右上の Get Started をクリック

<img width="2559" height="1373" alt="image" src="https://github.com/user-attachments/assets/8d135338-007a-4b2b-ac2f-33a539df09bb" />


#### ② サインアップ方法を選ぶ

 - GitHub連携が一番楽（後で必ずGitHubを使うので）

#### ③ GitHub認証（初回のみ）

- GitHubにログイン、Renderへのアクセス許可を Authorizeにする。

- この時点で公開/非公開はどちらでもOK

#### ④ Renderアカウント作成完了 

- 認証が終わると、自動で RenderDashboard に移動。
