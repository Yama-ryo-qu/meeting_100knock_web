**課題14.**


| ステータスコード | 分類 | 意味 |
| :--- | :--- | :--- |
| **200 OK** | 2xx (成功) | リクエストが正常に処理されたことを示す。 |
| **301 Moved Permanently** | 3xx (リダイレクト) | リクエストされたリソースが恒久的に新しい場所に移動したことを示す。 |
| **400 Bad Request** | 4xx (クライアントエラー) | クライアント（ブラウザなど）から送られたリクエストの構文が間違っているなど、クライアント側のエラーを示す。 |
| **404 Not Found** | 4xx (クライアントエラー) | リクエストされたリソース（ページやファイル）がサーバー上に見つからなかったことを示す。 |
| **500 Internal Server Error** | 5xx (サーバーエラー) | サーバー内部で予期しないエラーが発生し、リクエストが処理できなかったことを示す。 |
